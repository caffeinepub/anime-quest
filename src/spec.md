# Specification

## Summary
**Goal:** Add comprehensive game instructions and transform the UI to match Japanese/Korean anime game aesthetics while maintaining English language.

**Planned changes:**
- Create a tutorial/onboarding system explaining character unlocking, riddles, stories, and progress tracking
- Redesign UI with polished anime-style effects, refined typography, modern gradients, and professional spacing
- Add visual indicators, tooltips, and hover states throughout the interface
- Enhance character cards with particle effects, glow animations, premium borders, and dynamic states
- Improve home page with engaging hero section and clearer visual hierarchy

**User-visible outcome:** Players will have clear guidance on how to play through an accessible tutorial system, and experience a significantly more polished, professional anime-style interface with premium visual effects and intuitive navigation.
